2.padding

哈哈哈哈

给盒子指定padding后

  • 内容和边框之间有了距离，添加了内边距
  • 盒子变大

简写

值的个数 表达意思
1个值 padding:上，下，左，右内边距
2个值 padding：上下内边距，左右内边距
3个值 padding：上内边距，左右内边距，下内边距
4个值 padding：上内边距，右内边距，下内边距，左内边距

盒子的实际宽度：内容宽度 高度+内边距+边框

不改变盒子大小解决办法：内边距一定要给的，只能改变内容宽度，width-内边距

如果盒子里的内容没有宽度，padding不会撑开盒子

3.margin

简写写法与padding完全相同

盒子水平居中：左右外边距auto

文字居中和盒子居中的区别

  • 文字水平居中是text-align:center;而且可以让行内元素和行内块元素水平居中对齐。
  • 盒子水平居中是左右margin设为auto
text

4.注意

1）为了浏览器兼容性，首先应清楚元素的内外边距（重要）

    *{
        margin:0;
        padding:0;
    }

2）行内元素为了照顾兼容性，尽量只设置左右内外边距，不要设置上下内外边距

5.外边距合并

1）相邻块元素垂直边距会合并：上面的盒子有下外边距，下面的盒子有上外边距，他们之间的垂直间距不是两个边距之和而是去两个中较大的值。

解决方案：尽量只给一个盒子添加margin值

2）嵌套元素垂直外边距的合并（塌陷）：父元素没有上内边距及边框，父元素的垂直外边距会和子元素的垂直边距合并，合并后的外边距为两个中较大的值。

解决方案：①给父元素定义上边框；②给父元素定义上内边距；③给父元素添加overflow:hidden;
