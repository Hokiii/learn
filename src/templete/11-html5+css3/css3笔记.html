1.CSS3 选择器

CSS3中新增了3种选择器

  • 属性选择器
  • 结构伪类选择器
  • 伪元素选择器

1.1 属性选择器

语法： 
    选择器[属性名 = / ^= / $= / *= "val"] {
    }
     = 以val为属性值
    ^= 属性值以val开头
    $= 属性值以val结尾
    *= 属性值包含val

    注意：属性选择器权重为10

1.2 结构伪类选择器

    选择器：first-child/last-child/nth-child(数字/公式，如2n表示偶数) {
    }
    注意：结构伪类选择器权重为10

1.3 伪元素选择器

    选择器::before / ::after {
        content ：content值；
    }
    注意：before和after必须有content属性
          before和after创建一个行内元素，因为在dom里看不见创建的元素，所以称为伪元素
          伪元素选择器权重为1

2.CSS3 2D转换

转换（transform）是CSS3的特征之一，可以实现元素的位移，旋转，缩放等

转换（transform）可以理解为变形

  • 移动：translate
  • 旋转：rotate
  • 缩放：scale

2.1 移动(translate)

translate改变元素在页面中的位置，类似定位

语法

    transform: translate(x,y);或者分开写
    transform: translateX(n);
    transform: translateY(n);

注意

  • translate最大的优点：不会影响其他元素的位置
  • 它的百分比单位是相对于自身的
  • 对行内标签没有效果

2.2 旋转(rotate)

2D旋转指的是让元素在二维平面中顺时针旋转或逆时针旋转

语法

    transform：rotate(度数);

注意

  • rotate里面放度数，单位是deg，如rotate(45deg)表示顺时针旋转45°
  • 角度为正时表示顺时针，为负时表示逆时针
  • 默认旋转中心是自身的中心，改变旋转中心：transform-origin：x y;也可以是方位名词
  • 经常与transition（过渡）效果配合使用

2.3 缩放(scale)

缩放(scale)是指放大和缩小

语法

    transform：scale(x,y);

    注意：transform：scale(2,2);宽高都放大了两倍，transform：scale(0.5,0.5);缩小为原来的1/4
          transform：scale(2);只写一个参数相当于两个参数都一样
          默认以中心点缩放，不影响其他盒子，可以转换中心点缩放

2.4 2D转换综合写法

同时使用多个转换用空格隔开，tranform: translate() rotate() scale()...等

顺序会影响转换的效果，旋转会改变坐标轴的方向，当同时有位移和其他属性是，将位移放到最前面。

3.CSS3 动画（animation）

3.1 制作动画的步骤

制作动画分两步

  1. 定义动画
  2. 调用动画

（1）用keyframes定义动画

    @keyframes 动画名称{
        动画序列（关键帧的顺序）：
        初始状态
        0% {
        }
        ...可以有很多关键帧
        结束状态
        100% {
        }
    }

（2）元素使用动画

    选择器 {
        animation-name: 动画名称；
        animation-duration: 持续时间；
    }

3.2 动画常用属性

3.3 动画属性简写

animation: 动画名称 持续时间 运动曲线 何时开始 播放次数 是否反方向 动画起始或结束的状态;

例：animation: move 2s ease 1s infinite alternate backwards;
  • 简写属性里不包括animation-play-state
  • 暂停动画：animation-play-state：paused;经常和鼠标经过等其他配合使用
  • 想要动画走回来而不是直接跳回来，animation-direction: alternate;
  • 盒子动画结束后，停在结束位置，animation-fill-mode: forwards;

4.CSS3 3D转换

4.1 主要知识点

  • 3D位移:translate3d(x,y,z)
  • 3D旋转：rotate3d(x,y,z)
  • 透视：perspective
  • 3D呈现：transform-style

4.2 3D移动 translate3d

3D移动在2D移动的基础上多加了一个可以移动的方向，就是z轴方向

- transform: translateZ(100px);仅仅在Z轴移动（向外移动）100像素（注意：translateZ一般跟px做单位）
- transform: translate3d(x,y,z);其中x,y,z分别指要移动的轴的方向的距离（注意：x,y,z不能省略，如果没有就写0）

4.3 透视 perspective

  • 透视也称为视距：视距就是人的眼睛到屏幕的距离
  • 近大远小
  • 透视的单位是像素
注意：透视写到被观察元素的父盒子上，透视（视距）越小，被观察元素越大

4.4 3D旋转 rotate3d

3D旋转可以让元素在三维平面内沿着x轴，y轴，z轴或者自定义轴进行旋转

- transform: rotateX(45deg);仅仅绕X轴旋转45°
- transform: rotateY(45deg);仅仅绕Y轴旋转45°
- transform: rotateZ(45deg);仅仅绕Z轴旋转45°
- transform: rotate3d(x,y,z,deg);其中x,y,z分别指要移动的轴的方向的距离（注意：x,y,z不能省略，如果没有就写0）

4.5 3D呈现 transform-style

  • 3D呈现 transform-style控制子元素是否开启三维立体环境
  • transform-style: flat;子元素默认不开启3D空间
  • transform-style: preserve-3d;子元素开启3D空间
  • 代码写在父级，但影响的是子盒子

5.浏览器私有前缀

浏览器私有前缀是为了兼容老版本的写法，比较新版本的浏览器无需添加

私有前缀

  • -moz-：Firefox浏览器私有前缀
  • -ms-：IE浏览器私有前缀
  • -webkit-：safari、chrome浏览器私有前缀
  • -o-：Opera浏览器私有前缀
