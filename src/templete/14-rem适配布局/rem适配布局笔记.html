rem适配布局

1. rem基础

em，rem都是相对长度单位

  • em是相对于父元素的字体大小来说的
  • rem是相对于html元素的字体大小来说的
  • rem的优点就是可以通过修改html里面的文字大小来改变页面中元素的大小，可实现整体控制

2. 媒体查询

媒体查询（Media Query）是CSS3新语法，@media可以针对不同的屏幕尺寸设置不同的样式

语法： 
    @media mediatype and/not/only (media feature) {
        CSS Code;
    }
    - 用@media开头注意@符号
    - mediatype 媒体类型：all所有类型/print打印设备/screen电脑屏幕，平板电脑，智能手机
    - 关键字将媒体类型或多个媒体特性连接在一起作为查询的条件：and“且”将多个媒体特性连在一起
                                                              not“非”排除某个媒体类型，可以省略
                                                              only指定某个特定的媒体类型，可以省略
    - media feature媒体特性必须有小括号包含：width定义输出设备中页面可见区域的宽度
                                             min-width定义输出设备中页面最小可见区域的宽度
                                             max-width定义输出设备中页面最大可见区域的宽度

媒体查询还可以用于引入资源,即根据不同的屏幕尺寸调用不同的CSS文件

3. Less基础

Less是一门CSS预处理语言，它扩展了CSS的动态特性，常见的CSS预处理器有：Sass，Less，Stylus，Less中文网址：http://lesscss.cn

3.1 Less嵌套

less中，子元素的样式直接写到父元素中

例 
    css: header a:hover {}             less: header{ a{ &:hover { }} }
如果有伪类，交集选择器，伪元素选择器，内层选择器的前面需要加“&”符号

3.2 less运算

  • 运算符的左右两侧必须有一个空格
  • 如果两个数参与运算，只有一个数有单位，则最后的结果就以这个单位为准
  • 如果两个数参与运算，两个数都有单位，且单位不一样，最后的单位以第一个为准

4. rem适配方案

  • 让一些不能等比自适应的元素达到当设备尺寸发生改变时，等比例适配当前设备
  • 使用媒体查询根据不同设备按比例设置HTML的字体大小，然后页面元素使用rem做尺寸单位，当HTML字体大小变化，元素尺寸也会发生变化，从而达到等比例缩放的适配

rem适配方案技术使用

技术方案1：less + 媒体查询 + rem
技术方案1(推荐)：flexible.js + rem
步骤：
  • 先选一套标准尺寸，以750px为准
  • 用屏幕尺寸/划分的份数=html里面的文字大小，但不同屏幕下得到的文字大小时不一样的
  • 最后不同屏幕尺寸下页面元素的rem值=页面元素在标准尺寸下的像素值/html里面的字体大小
