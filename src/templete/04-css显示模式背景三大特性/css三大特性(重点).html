CSS三大特性

1.CSS层叠性

概念

  • 层叠性是指多种css样式的叠加;
  • 是浏览器解决冲突的一个能力，如果一个属性通过两个相同的选择器设置在同一个元素上，那么这时一个属性就会将另一个属性覆盖掉。

原则

  • 样式冲突，遵循就近原则，哪个样式离结构近就执行哪个样式；
  • 样式不冲突，不会层叠。

2.CSS继承性

概念

  • 子标签会继承父标签的某些样式，如文本颜色和字号；
  • 想要设置一个可继承的属性，只需将它应用于父元素即可。

注意

  • 恰当的使用继承，可以简化代码，降低css的复杂性。比如有很多子级孩子都需要某种样式，可以给父级指定一个，孩子继承即可；
  • 子元素可以继承父元素的样式（text-,font-,line-,color）。

3.CSS优先级（重点）

概念：定义css样式时，经常出现两个或更多规则应用在同一元素上

  • 选择器相同，则执行层叠性，即“就近原则”；
  • 选择器不同，就会出现优先级问题

1).权重计算公式

标签选择器 计算权重公式
继承或*（通配符选择器） 0,0,0,0
每个元素（标签选择器） 0,0,0,1
每个类，伪类 0,0,1,0
每个ID 0,1,0,0
每个行内样式 1,0,0,0
每个！important重要的 ∞无穷大
  • 权重值从左至右，左边最大。

2).权重叠加

我们经常用交集选择器，后代选择器等，是有多个基础选择器组合而成，此时就会出现权重叠加，就是简单的加法计算。

  • div ul li ------> 0,0,0,3
  • .nav ul li ------> 0,0,1,2
  • a:hover ------> 0,0,1,1
  • .nav a ------> 0,0,1,1

注意

  • 数位之间没有进制

3)继承的权重是0

修改样式，需要看标签是否被选中

  • 如果选中，则以权重叠加进行计算
  • 如果未选中，那么权重是0，因为继承的权重是0

注意

有2个特殊标签a链接和h标题他们浏览器有默认的样式，继承的权重为0，所以我们还要单独给链接和标题一个样式
