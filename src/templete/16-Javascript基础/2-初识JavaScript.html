初识Javascript

1. Javascript是什么

Javascript是一种运行在客户端的脚本语言,不需要编译,运行过程由js解释器(js引擎)逐行解释并执行,现在也可以给予Node.js技术进行服务器端编程.

2.浏览器执行Javascript的原理

浏览器分为两部分:渲染引擎和JS引擎

  • 渲染引擎:解析HTML和CSS,俗称内核,如blink,webkit
  • JS引擎:JS解释器,浏览器本身并不会执行JS代码,而是通过内置的JS引擎来执行JS代码. JS引擎执行代码时逐行解释每一句源码(转换成机器语言)然后由计算机执行,所以js语言归为脚本语言,会逐行解释运行.

3. Javascript的组成部分

JavaScript
ECMAScript : Javascript语法(基础)
DOM : 页面文档对象模型(API)
BOM : 浏览器对象模型(API)
